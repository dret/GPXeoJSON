GeoJSON-X Well-Known Extensions
===============================

Well-Known Values
-----------------

| Value | Description | Source |
| ----- | ----------- | ------ |
| time | A time stamp (defined as xsd:dateTime in GPX 1.1) for an individual geographic point. | GPX 1.1 |
